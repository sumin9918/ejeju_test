## 브런치
- 로컬 브랜치 : 내 컴퓨터에 있는 브랜치
- 원격 브랜치 : github에 있는 브랜치

#### 브랜치 이동
```
git checkout 이동할 브랜치명
```

#### 브랜치 삭제법
- 반드시 변경 사항이 없을 때에만 하세요

- 먼저 dev 브랜치 이동
```
git checkout dev
```
- 로컬 브랜치 삭제
```
git branch -d 브랜치명
git branch -D 브랜치명 # 강제 삭제
```
- 원격 브랜치 삭제
```
git push origin --delete 브랜치명
```

#### 브랜치 조회
- 로컬 브랜치 목록 확인
```
git branch
```
- 원격 브랜치 목록 확인
```
git branch -r
```
